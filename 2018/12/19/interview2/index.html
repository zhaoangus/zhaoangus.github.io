迭代器

for-of

for-of是es6中新增，得到的是一个可以保证顺序的迭代（与for以及for-in不同）。操作的对象具有一个Symbol.iterator属性，该属性被for-of使用。

内置迭代器

String,Array,Map,Set都内置迭代器。

解构赋值

var hello = 'world';
var [first, second, ...rest] = [...hello];
console.log(first, second, rest); // w o ["r","l","d"]

generator函数

它是一个可以暂时退出，并且稍后重新进入继续执行的函数。在多次的进入中，它的上下文（绑定的变量）是会被保存的。generator函数自身就是一个迭代器。

Promise

语法

new Promise( function(resolve, reject) {...} /* executor */ );

Promise对象有三种状态

  1. pending: 等待中，或者进行中，表示还没有得到结果
  2. resolved(Fulfilled): 已经完成，表示得到了我们想要的结果，可以继续往下执行
  3. rejected: 也表示得到结果，但是由于结果并非我们所愿，因此拒绝执行

在Promise对象的构造函数中，将一个函数作为第一个参数。而这个函数，就是用来处理Promise的状态变化。

then

Promise对象中的then方法，可以接收构造函数中处理的状态变化，并分别对应执行。then方法有2个参数，第一个函数接收resolved状态的执行，第二个参数接收reject状态的执行。
then方法的执行结果也会返回一个Promise对象。

then方法可以链式调用。

p1
  .then(step1)
  .then(step2)
  .then(step3)
  .then(
    console.log,
    console.error
  );

上面代码中，p1后面有四个then，意味依次有四个回调函数。只要前一步的状态变为fulfilled，就会依次执行紧跟在后面的回调函数。

最后一个then方法，回调函数是console.log和console.error，用法上有一点重要的区别。console.log只显示step3的返回值，而console.error可以显示p1、step1、step2、step3之中任意一个发生的错误。举例来说，如果step1的状态变为rejected，那么step2和step3都不会执行了（因为它们是resolved的回调函数）。Promise 开始寻找，接下来第一个为rejected的回调函数，在上面代码中是console.error。这就是说，Promise 对象的报错具有传递性。

注意点

  1. Promise 的回调函数属于异步任务，会在同步任务之后执行，但在其他正常异步任务之前执行；
  2. Promise 在新建后立即执行，所以首先输出的是Promise。然后，then方法指定的回调函数，将在当前脚本所有同步任务执行完才会执行。
    `
    let promise = new Promise(function(resolve, reject) {
    console.log(‘Promise’);
    resolve();
    });

promise.then(function() {
console.log(‘resolved.’);
});

console.log(‘Hi!’);

// Promise
// Hi!
// resolved
`

跨域

为什么有跨域

浏览器的同源策略导致了跨域。
同源策略限制了从同一个源加载的文档或脚本如何与来自另一个源的资源进行交互。这是一个用于隔离潜在恶意文件的重要安全机制。

同源

如果两个页面的协议，端口（如果有指定）和域名都相同，则两个页面具有相同的源。
注：IE未将端口号加入同源策略的组成部分中。

同源策略

  1. 针对接口请求
  2. 针对DOM的查询
    对第一种，采用设置httpOnly使得无法通过js操作cookie

同源下接口请求的正确打开方式

  1. JSONP
  2. 代理服务器
  3. CORS

js改变与不改变原数组的方法整理

改变：

  1. push(返回添加元素后的数组长度)
  2. unshift(返回添加元素后的数组长度)
  3. pop(返回删除的元素)
  4. shift(返回删除的元素)
  5. reverse(返回逆序后的数组)
  6. splice(返回删除的元素组成的数组)
    不改变：
  7. concat(返回原数组和参数数组合并后的新数组)
  8. slice(返回指定数组的一个片段或子数组)
  9. map(函数返回值组成的新数组)
  10. filter(返回调用数组的一个子集)
