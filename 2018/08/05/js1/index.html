高阶函数

高阶函数也就是某个函数接收另一个函数作为参数（如回调函数），最近在学习中遇到了一些相关问题，决定整理出来。

sort

sort()作为一个高阶函数的使用方法，可以接收一个函数作为参数，该函数规定排序的顺序。

如果调用该方法时没有使用参数，将按字母顺序对数组中的元素进行排序，说得更精确点，是按照字符编码的顺序进行排序。要实现这一点，首先应把数组的元素都转换成字符串（如有必要），以便进行比较。

如果想按照其他标准进行排序，就需要提供比较函数，该函数要比较两个值，然后返回一个用于说明这两个值的相对顺序的数字。比较函数应该具有两个参数 a 和 b，其返回值如下：

  • 若 a 小于 b，在排序后的数组中 a 应该出现在 b 之前，则返回一个小于 0 的值。
  • 若 a 等于 b，则返回 0。
  • 若 a 大于 b，则返回一个大于 0 的值。

也就是说如果需要比较数字类型，可传入一个比较函数。

```
    var arr = [10, 20, 1, 2];
    arr.sort(function (x, y) {
    if (x < y) {
        return -1;
    }
    if (x > y) {
        return 1;
    }
    return 0;
    }); // [1, 2, 10, 20]
```

此时sort()方法传入的函数接收的两个参数分别代表每次排序比较时的两个数组项。

V8引擎的sort()方法在底层实现时采用了插入排序和快速排序两种算法，如果数组长度小于等于22采用插入排序，其他长度采用快速排序。

// In-place QuickSort algorithm.
// For short (length <= 22) arrays, insertion sort is used for efficiency.

sort()方法接收函数这种方法还可以较为方便的解决多维数组按照某一列元素进行排序的问题。今天遇到的一个实际问题如下所示：

```
(function(){
    var aqiData = [
    ["北京", 90],
    ["上海", 50],
    ["福州", 10],
    ["广州", 50],
    ["成都", 90],
    ["西安", 100]
    ];

    var saqiData = aqiData.sort(function(x,y){
        return y[1]-x[1];
    });
})();
```

排序后数组aqiData将根据第二列数字降序排列。
